# Pure Zakat Calculator

A modern, accurate, and easy-to-use Zakat calculator built with React, TypeScript, and Tailwind CSS.

## Features

-   **Real-time Metal Rates**: Fetches live gold and silver prices (USD) to calculate Nisab accurately.
-   **Multi-Currency Support**: Calculate Zakat in USD, INR, SAR, GBP, and EUR.
-   **Nisab Calculation**: Automatically determines the Nisab threshold based on live silver/gold prices.
-   **Asset Breakdown**: Input values for Gold/Silver, Cash, Investments, and Business Goods.
-   **Liability Deduction**: Deduct short-term debts to find net zakatable wealth.
-   **Zakat Result**: Instantly calculates the exact Zakat amount (2.5%) if obligated.

## Tech Stack

-   **Frontend**: React + TypeScript
-   **Styling**: Tailwind CSS + shadcn/ui
-   **Build Tool**: Vite

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/razinshaikh/pure-zakat.git
    cd pure-zakat
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```
